# react-antd-admin
基于react和antd的后台管理系统模版
### 文件介绍：
```
dist                打包文件
distBack            备份文件夹
public              html模版文件
src                 主程序
 ├──  api           接口文件夹
 ├──  comm          公共方法文件夹
 ├────── common.js  公共方法
 ├────── https.js   axios拦截器
 ├────── validate.js  自定义表单验证
 ├──  components    公共组件库文件夹
 ├──  modules       业务模版文件夹
 ├────── layout     基本框架内容
 ├────── error      错误提示页面
 ├──  store         redux存储方法
 ├──  style         静态资源文件夹
 ├──────  css       公共样式
 ├──────  img       图片
 ├──  App.js        主路由文件
 ├──  index.js      入口文件
antd.customize.less 配置antd自定义样式
clean-distBack.js   清除dist
craco.config.js     配置webpack
pages.config.js     公共参数配置
```
